== Description ==

Plugin where you can select what tags you want to remove from the head section of your wordpress site. If you have any suggestions, please drop us a line!

== Installation ==


1. Upload `doggus_clean_heads.zip` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
